    Windows stores user Credentials in SAM & SYSTEM, it stores them as NTLM hashes. Hashes are cryptographically secure. They only work on way, but that doesn’t mean they aren’t without faults. We will be using some pwdrump which is should be reinstalled on kali.

    cd /mnt/bast-vhd2/Windows/System32/config

    As mentioned above will be using PWDump which will extract the NTLM hashes from SYSTEM hive and SAM hive

    pwdump SYSTEM SAM
