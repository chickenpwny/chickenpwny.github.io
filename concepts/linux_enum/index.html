    Linux Enumeration

    I always start with nmap i use TCP Connect switch, then move to targeted scans. Then i run a full udp scan. While udp is being scanned. I will then move to gobuster, start manual enumeration. Things to look for are input fields of any kind, look for php parameters/actions in the source.

    nmap

    nmap -sT -p- -T5 -oA nmap/tcpconnect ipaddr
nmap -sC -sV -T4 -pport,port,port -oA nmap/targeted ipaddr
nmap -sU -p- -T5 -oA nmap/udp ipaddr

    Gobuster

    I try to start with the small list first gobuster can be rather slow, and time consuming so it fits in with manual enumeration.
