    GoBuster

    Go Buster

    Gobuster is a tool often used to enumerate web directories. It offers more power than dirb & dirbuster. You dont have to use the extension switch all the time and it may not be necessary. I would recommend limiting extensions to whats relevant.

    Enumerating Directories

    gobuster -u http://TARGET_IP:PORT -w /usr/share/wordlists/dirbuster/directory-list-lowercase-2.3-medium.txt -t 100

    -t for threads can go as high as 200, 100 is kind of to much 40 is perfect for multiple things going on.

    -s “200,204,301,302,307,404” errors to look for you can edit these

    -w medium or small

    -x extensions Linux php html txt doc Windows aspx asp php

    -fw force wild card

    Adding SessionID this can be easily overlooked.

    -c PHPSESSIONID=1324567890

    Enumerating Browser User Agent

    Burp Proxy User Agent (certain browser Sufarri, FireFox, Chrome, IE).

    gobuster -u http://localhost:3000 -w /usr/share/wordlists/dirbuster/directory-list-lowercase-2.3-medium.txt -t 39

    gobuster -u http://target:port -a ‘agent’ -w wordlists -t 100

    Adding multple extentions

    -x php,aspx,asp,html,doc,txt
